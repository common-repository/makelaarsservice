Verbindt uw Makelaarsservice account met uw WordPress website.

== Description ==

Toon uw makelaars objecten vanuit Makelaarsservice direct op uw website, zonder deze handmatig in te hoeven voeren!

Zodra u een WordPress token hebt ontvangen kunt u deze gebruiken met onze plug-in. Volg onderstaande installatie stappen
en uw objecten zijn zichtbaar binnen no-time!

== Installation ==

1. Installeer de plug-in via de WordPress plugin directory.
2. Activeer de plug-in in het plugin menu van WordPress.
3. Klik op de nieuw verschenen menuknop 'Makelaarsservice'
4. Vul uw Makelaarsservice token in en klik op 'Toevoegen'
5. Indien u een geldige token heeft opgegeven zal deze worden toegevoegd
en zullen uw objecten automatisch worden ge√Ømporteerd. Zodra dit is voltooid
verschijnt er een melding '.. succesvol bijgewerkt!'. Gefeliciteerd! Uw woningen
zijn nu toegevoegd.

De listing pagina is te vinden onder url '/objecten'.
Voeg deze url toe aan het navigatiemenu.

Individuele objecten verschijnen onder url '/objecten/{adres-plaatsnaam}'
